# TeamPetra
Building an innocent Java game

THE GAME

Two-player fighting game. 
Player 1 uses WAD, where AD - movement, W - block. F and G are punch and kick. 
Player 2 uses the arrow keys + K and L. Same logic - K - punch, L - kick. Enjoy!
